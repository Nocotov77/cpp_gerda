# Простой принтер

**Ограничение времени:** 1 секунда

**Ограничение памяти:** 64.0 Мб

**Ввод:** `test.cpp`

**Вывод:** стандартный вывод или `output.txt`

Вам предстоит реализовать функцию `void simple_printer`, которая выводит на экран фразу "Это сообщение было напечатано функцией".

## Пример

**Ввод**
```cpp
#include <iostream>
#include "main.cpp"

int main() {
	simple_printer();
	return 0;
}
```

**Вывод**
```
Это сообщение было напечатано функцией
